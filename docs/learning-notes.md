# Lesson 1

## Topics Learned

- Difference between ServerSocket and Socket
- TCP Connections
- Blocking methods
- accept()
- readLine()
- InputStream
- OutputStream
- HTTP Request structure
- HTTP Response structure
- Browser request lifecycle

## Key Takeaways

- A ServerSocket listens for incoming connections.
- A Socket represents communication with one client.
- HTTP is a text-based protocol built on TCP.
- Browsers send multiple requests for a single page.
