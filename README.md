# 01 - Computer Networks

## Overview

This project is part of my Backend Engineering Roadmap.

The objective is to understand how web servers work internally by implementing networking concepts from scratch in Java instead of relying on frameworks like Spring Boot or Tomcat.

---

## Concepts Covered

- TCP/IP Communication
- Client-Server Architecture
- ServerSocket
- Socket
- Blocking I/O
- InputStream
- OutputStream
- HTTP Request
- HTTP Response
- Request-Response Lifecycle

---

## Features Implemented

- Start a TCP server on port 8080
- Accept browser connections
- Read raw HTTP requests
- Send HTTP responses
- Render HTML in a browser

---

## Technologies

- Java 21
- IntelliJ IDEA
- Git

---

## Future Enhancements

- Parse HTTP headers
- Build an HTTP request parser
- Implement routing
- Handle multiple clients using threads
- Build a mini web server

## Learning Goal

This repository focuses on understanding the fundamentals behind backend frameworks before using Spring Boot.
